Mutual Fund Data Cleaning and Analysis

Overview
This project focuses on cleaning and analyzing mutual fund data using Python, Pandas, and NumPy.

Tools Used
- Python
- Pandas
- NumPy
- Google Colab

Steps Performed
- Imported dataset
- Handled missing values
- Removed duplicate records
- Standardized column names
- Cleaned category values
- Converted launch date column into datetime format
- Created launch year and launch month features
- Created AUM classifications using NumPy
- Exported cleaned dataset

Key Learning
Learned practical data cleaning and feature engineering using financial datasets.
